# Customer Churn Analysis & Prediction

## 📊 Overview
Analyzed customer churn data using SQL, Power BI, and Machine Learning to identify key factors influencing customer retention.

## 🧰 Tools Used
- MySQL (Data Analysis)
- Power BI (Dashboard)
- Python (Machine Learning)

## 📈 Key Insights
- Month-to-month contracts have highest churn
- High monthly charges increase churn risk
- New customers are more likely to leave
- Fiber optic users show higher churn
- Lack of tech support increases churn

## 🤖 Machine Learning
Built a Logistic Regression model to predict churn with ~80% accuracy.

## 🚀 Outcome
Delivered actionable insights to improve customer retention and reduce churn.
